# Hotel web app

This project was created as a practice for React and Redux. You can check [demo](http://scots-hotel-12387.herokuapp.com/).

## Available Scripts

In the project directory, you can run:

### `yarn startReact`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\

<b>For development server</b>

### `yarn server`

You have avalible Resources:
[All Rooms](http://localhost:4000/rooms)
[Room search](http://localhost:4000/rooms/id)
[All bookings](http://localhost:4000/booking)
[Room bookings](http://localhost:4000/rooms/roomID/booking)
